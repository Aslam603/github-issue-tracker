## Answer to the question no-2

## Spread operator: 
The spread operator is used to copy or merge arrays and objects.Spread operator(...)
is used to spread elements of an array or object into another array or object.Such as:

const arr1 =[1,2,3];

const arr2 =[...arr1,4,5]
